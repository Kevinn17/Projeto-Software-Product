🚗 É uma aplicação web robusta desenvolvida em Django para o gerenciamento de inventário de veículos. O sistema permite o cadastro de marcas e modelos, controle de estoque automatizado via Signals e uma interface administrativa completa.

🚀 Funcionalidades
Gestão de Veículos: CRUD completo (Criação, Leitura, Atualização e Exclusão) de carros.

Controle de Marcas: Cadastro centralizado de fabricantes.

Inventário Automatizado: Uso de Django Signals para atualizar a contagem e o valor total do estoque em tempo real a cada alteração no banco.

Busca Inteligente: Filtro dinâmico de veículos por modelo.

Segurança: Acesso restrito a usuários autenticados para operações de escrita e exclusão.

Bio Automática: Geração de descrição padrão para novos veículos via pre_save.

🛠️ Tecnologias Utilizadas

Python 

Django 

PostgreSQL 

HTML/CSS
